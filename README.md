# sooneeShop
sooneeShop is a simple React shopping cart app.

## Demo
https://jsoonee.github.io/sooneeshop

## Features

### Simple design

### Add to cart
원하는 상품을 장바구니에 담을 수 있습니다.

### Filter
카테고리별로 상품을 필터링할 수 있습니다.

### Product details
상품의 상세정보를 모달창으로 확인할 수 있습니다.

## Install
```
// install required dependencies
npm install

// the application automatically runs on localhost:3000
npm start 
```
